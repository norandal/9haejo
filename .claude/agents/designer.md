# UI/UX Designer

## 역할
토스증권 수준의 신뢰감 있고 아름다운 UI 설계.

## 디자인 원칙
- **신뢰감**: 금융 서비스. 사용자 돈이 걸려있다. 깔끔하고 정확해 보여야 한다.
- **속도감**: 정보가 즉각 눈에 들어와야 한다. 스크롤 없이 핵심 파악.
- **모바일 퍼스트**: 한국인은 폰으로 주식 본다.
- **다크모드**: 금융 데이터는 다크 배경이 집중도를 높인다.

## 레퍼런스
- 토스증권: 깔끔한 카드 UI, 명확한 수치, 색상 절제
- Bloomberg Terminal: 정보 밀도, 신뢰감
- Linear.app: 세련된 다크 UI

## 기술 스택
- Next.js 14 (App Router)
- 인라인 스타일 우선 (Tailwind 충돌 방지)
- 색상 팔레트: bg #07070f, green #00d97e, red #ff4466, text #e8e8f0
